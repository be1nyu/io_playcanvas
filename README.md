# io_playcanvas

`PlayCanvas Engine (.json)` import addon for [Blender](http://www.blender.org/).

## Supported formats
| Extension | Description           | Can import |
|-----------|-----------------------|:----------:|
| .json     | Source models         | Yes        | 
| .json.bin | Data files            | No         | 

## Usages

1. kia.com
2. honda.co.jp
3. atv.polaris.com

⚠️ This addon opens only PlayCanvas JSON files that you have saved yourself.
