# io_playcanvas

PlayCanvas (.json) Import add-on for Blender 

## Installation

1. Download [the repository](https://github.com/be1nyu/io_playcanvas/releases)
2. In Blender, go to Edit > Preferences, and navigate to the "Add-ons" tab. In here click install and select the .py file.
3. Done
